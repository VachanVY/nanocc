# Unit Tests

This directory contains unit tests for nanocc components.

## Structure

Tests are organized by component:
- `Lexer/` - Lexer unit tests
- `Parser/` - Parser unit tests
- `Sema/` - Semantic analysis unit tests
- `IR/` - IR generation unit tests
- `Codegen/` - Code generation unit tests
